The user handling functions allow you to get keyboard and mouse input from the user, either for turn by turn games (the function wait until the user press a key or a mouse button), or real time games (non blocking function).
WARNING : those functions also handle screen redraw event, so TCODConsole::flush function won't redraw screen if no user input function is called !
